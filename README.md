# OnlineAdmissionTest
It is a mcq-test based web application created using Java Spring Boot. It is generally developed to understand the Spring MVC and Spring Boot framework's features.


# Project Features
* Login Page
* Questions Page
* Scorecard Page
* Final Results Page
